## System Requirements:

1. `Node.js 16.8` or later.
2. `macOS`, `Windows (including WSL)`, and `Linux` are supported.

## Run in your Local Machine :

1. `Open Project in VSCode`

2. `Open Terminal`

3. `cd MAISON-ENSEMBLE-WEB`

4. `npm install`

5. `npm run dev`
